Nosso trabalho consiste em um sistema de cadastro de produtos de maquiagem (blush, base, gloss, batom). O sistema permite realizar o cadastro do produto, editar, excluir e a listagem, além da possibilidade de filtrar os produtos por marca.

Alunas: Bianca Magdalena e Vitória Sá

As tecnologias utilizadas foram o TypeScript, Node js, MySQL e o Vite.

Primeiro execute o SQL para criar o banco de dados e as tabelas com os dados
Depois, abrir o terminal na pasta do backend e executar os comandos "npm install" e depois "npm run dev"
Depois, abrir um outro terminal na pasta de frontend e executar os mesmos comandos
Dar ctrl e apertar o link para acessar o vine com o projeto feito
